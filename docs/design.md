# 异世界开拓者 · 设计文档 v4 (发行版)

## 核心定位
**章节制叙事驱动 · 轻量手游 · 双模式切换 · 角色进化**

不是沙盒建造，不是肉鸽远征。是"异世界转生"式的故事——每一章到一个新地方、遇到新事件、角色也在变。
你可以在任何章节暂停，深挖经营。

## 游戏模式

### 🏠 家园模式
- 固定2×3建筑槽位，解锁=成长
- 6种建筑：小屋/仓库/工具台/熔炉/菜园/观星台
- 经营触发的奇异事件（而非无限种田）
- NPC交互（莉莉）→ 推进剧情

### 🗺️ 远征模式
- 节点地图 → 选择方向 → 遭遇事件 → 做选择 → 得收获
- 事件池随章节和建筑等级扩展
- 选择塑造叙事风味（不影响推进，影响世界反应）

## 章节规划

### 序章 · 苏醒 (Days 1-3)
- 醒来 → 莉莉 → 首次远征 → 夜空 → 梦境 → 角色初进化 → 再次远征 → 建造
- 核心悬疑建立：第47个、前任领主、边缘崩塌、无形之声

### 第一章 · 林中之声 (Days 4-7)
- 观星台建成 → 发现森林异常
- 镜面蝶群、记忆水母、虚空商贩
- 声音之谜深化
- 第一个大的角色变化
