## 重大方向调整

用户否定了星露谷/泰拉瑞亚式的自由沙盒方向（WASD跑图+自由建造+制作树），也否定了纯肉鸽远征。最终确定核心：

**章节制故事驱动 + 双模式（家园/远征）+ 角色进化 + 叙事沉浸**

## 新架构

### 核心循环
```
家园界面 (固定建筑槽位) ←→ 远征界面 (节点地图+事件)
```

### 游戏骨架
- **章节制**：序章→第一章→...每个章节=一片新区域+独立事件+阶段性故事
- **数据驱动**：章节和事件都是JSON配置，加新内容不用改核心逻辑
- **角色进化**：故事事件触发外观/能力/种族变化，变化后反向影响探索

### 技术架构
- 横屏优先，响应式适配
- Canvas 2D 像素渲染（家园场景+远征地图）
- HTML/CSS 对话面板（保证文字渲染清晰+触屏友好）
- localStorage v4存档（15秒自动保存）
