## 游戏流程
1. 采集木头/石头 → 建造工具台
2. 采集带红点的矿石(铁矿石) → 建造熔炉
3. 制作木板/铁锭 → 建造菜园/升级建筑
4. 探索东部 → 发现镜面蝶群/晶壳蜗牛/记忆水母
5. 收集资源 → 升级建筑 → 领地等级提升
6. 虚空商贩随机出现 → 静默交易
