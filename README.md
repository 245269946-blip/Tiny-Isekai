# 🌍 异世界开拓者 (Isekai Pioneer)

> *一个普通人类，被扔进一个物理法则都不太对劲的世界。你一边努力活下去，一边发现这个世界的奇妙——然后把一片荒地，变成"家"。*

---

## 🎮 游戏简介

**类型：** 像素风模拟经营 + 轻RPG  
**平台：** 浏览器直接运行（单HTML文件），PC+手机自适应  
**技术：** 纯前端 Canvas 渲染，localStorage 存档，零外部依赖  

---

## 🌀 三重核心张力

| 支柱 | 玩家感受 | 驱动 |
|------|---------|------|
| 🌀 奇异感 | 「天哪这是什么」 | 好奇心 |
| 🗡️ 探险感 | 「能活着回去吗」 | 紧张与贪心 |
| 🏡 安稳感 | 「我回来了」 | 归属与成就 |

## 🚧 开发分期

- **第一期 MVP** — 主界面 + 3种建筑 + 基础探索 + 资源循环
- **第二期** — 完整建筑 + 冒险者系统 + 多区域 + 图鉴
- **第三期** — 季节系统 + 随机事件 + 音效 + 数值平衡
